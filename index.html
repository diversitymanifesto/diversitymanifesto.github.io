Diversity and Inclusion Manifesto

"It's like the Agile Manifesto, but for humans."

One of the biggest challenges facing the Tech Industry, and in particular the Games Industry, today is a lack of Diversity and Inclusion (D&I).

The goal of this manifesto is to create a set of guidelines and best practices which will help everyone level up. Through sharing our successes and failures we can help create a more inclusive, diverse and welcoming environment for the next generation.

We will be contacting a number of studios and games publishers over the next 12 months to gather details of their D&I efforts. We'll also be gathering feedback from leaders in this field. This is a community effort in which everyone wins if we work together.

If you would like to get involved then drop a mail to : info@diversitymanifesto.org

Please help us spread the word about this initiative.

Thank you.
